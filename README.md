# Color Clock

A simple dynamic clock built with React, Vite, and date-fns.

## Features

- Displays the current date and time in a readable format
- Updates every second using React state and `useEffect`
- Uses `date-fns` for date formatting
- Includes colorful styling with a live accent color

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Notes

- The clock uses the `format` function from `date-fns`
- The current time and date are updated continuously in the UI
- Styling includes color beyond black and white for a vivid dashboard widget look
